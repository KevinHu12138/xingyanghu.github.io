Autonomous Mobile Robots

Project Type: Control Systems / UAS / Robotics Coursework

Technology Stack: Python, Feedback Control, Sensor Fusion, UAS Simulation

This project focuses on designing and implementing a feedback control algorithm for the position stabilization of an unmanned aerial system (UAS). The objective is to process sensor measurements, such as GPS data, and generate motor commands that enable the drone to maintain or achieve a target position in space.

In this work, I explored controller design for stability and responsiveness, evaluated the effect of sensor noise, and implemented the logic to run on a simulated aerial platform. The experience strengthened my understanding of control theory and practical tuning of UAV controllers in noisy environments.
